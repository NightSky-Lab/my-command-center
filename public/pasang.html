1 Sediakan Data (Supabase)

Hanya satu fail untuk di-RUN. Mudah.

  1. Pergi supabase.com → Start your project → daftar (Google).
  2. New Project → Region Singapore → Create → tunggu ~2 minit.
  3. Menu kiri → SQL Editor → + New query.
  4. Klik butang ini → Ctrl+A (pilih semua) → Ctrl+C (salin):
    📄 Buka fail data (setup.sql)
    Tampal ke SQL Editor → klik RUN ▶ (tunggu "Success").
  5. Klik butang hijau "Connect" (atas) → App Frameworks → Next.js. Salin 2 nilai (untuk Langkah 2).

2 Terbitkan Laman (Vercel)

Satu klik — salinan dibuat ke akaun anda automatik.

▲  Deploy ke Vercel
  1. Log masuk guna GitHub → klik Authorize/Continue (kalau diminta).
  2. Bila nampak Environment Variables → tampal 2 nilai dari Langkah 1.
  3. Klik Deploy → tunggu ~2 minit → 🎉 dapat link!

✓ Langkah Akhir

  1. Buka link → Log Masuk dengan kata laluan: mypi2025
  2. Tukar kata laluan di Profil → Keselamatan.
  3. Klik "Panduan Ringkas" dalam app untuk belajar cara guna setiap modul.

❓ Tersangkut?

Hubungi penjual anda untuk bantuan, atau minta mereka pasangkan terus. Kebanyakan masalah ialah kunci Supabase tersalah tampal — semak di Vercel → Settings → Environment Variables.
