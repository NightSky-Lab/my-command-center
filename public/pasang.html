Kenapa perlu daftar 3 akaun? 🤔

Supaya sistem anda berfungsi & data selamat. Setiap satu percuma — daftar guna butang "Continue with Google" (10 saat). Anda tak perlu faham teknikalnya.

🗄️

▲
Vercel — beri laman anda alamat (link) di internet supaya boleh dibuka.
🐙
GitHub — "kunci" untuk log masuk Vercel. Cuma daftar sekali.
✅ Semua percuma & selamat. Tiada bayaran, tiada kad kredit. Data 100% milik anda.

3 Langkah Mudah

Ikut satu-satu. Tak sampai 5 minit! 👇

1

Buat tempat simpan data

⏱️ ±2 minit · Supabase
  1. Buka supabase.com dan daftar (guna Google).
  2. Klik New Project. Pilih region Singapore, klik Create, tunggu ~2 minit.
  3. Di menu kiri, buka SQL Editor, klik New query.
  4. Tekan butang di bawah, pilih semua (Ctrl+A), salin (Ctrl+C), tampal ke SQL Editor, klik RUN:
📄 Buka Fail Data
  1. Klik butang Connect (di atas) → pilih Next.js → salin 2 kunci yang dipaparkan (kita guna di Langkah 2).
2

Terbitkan laman anda

⏱️ ±2 minit · Vercel
  1. Tekan butang Deploy di bawah.
  2. Log masuk guna GitHub (klik Authorize jika diminta).
  3. Di ruangan Environment Variables, tampal 2 kunci dari Langkah 1.
  4. Klik Deploy dan tunggu ~2 minit. 🎉 Anda dapat link laman anda!
▲ Deploy ke Vercel
💡 Pastikan 2 kunci ditampal betul (tiada ruang kosong di hujung).
3

Masuk & letak ikon app

⏱️ ±1 minit
  1. Buka link anda. Log masuk dengan kata laluan: mypi2025
  2. Tukar kata laluan di Profil → Keselamatan.
  3. Pasang sebagai app supaya senang buka (tekan ikon, terus masuk):
💻 Komputer: klik ikon Install (⊕) di bar alamat → Install.
📱 Android: menu ⋮ → Add to Home screen.
🍎 iPhone: butang Share ⬆️ → Add to Home Screen.
🎉 Selepas ini, cukup tekan ikon → terus masuk sistem. Klik "Panduan Ringkas" dalam app untuk belajar cara guna.
